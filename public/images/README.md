# 📸 Bilder für Sonnenhof Website

## 📁 Ordnerstruktur

```
images/
├── hero/                    → Hauptbilder für Startseite
├── ferienwohnungen/         → Fotos der 5 Ferienwohnungen
│   ├── ammersee/
│   ├── utting/
│   ├── andechs/
│   ├── herrsching/
│   └── diessen/
├── zimmer/                  → Fotos der Zimmer
│   ├── doppelzimmer-balkon/
│   ├── doppelzimmer/
│   └── einzelzimmer/
├── allgemein/               → Gebäude, Eingang, Parkplatz, Teeküche
├── umgebung/                → Ammersee, Herrsching, Berge
└── details/                 → Schöne Detail-Aufnahmen
```

## 📝 Benennungskonvention

**Format:** `[zimmer-name]-[nummer]-[bereich].jpg`

**Beispiele:**
- `ammersee-01-wohnbereich.jpg`
- `utting-02-kueche.jpg`
- `dz-balkon-01-zimmer.jpg`

## ⚙️ Technische Anforderungen

- **Format:** JPG
- **Hero-Bilder:** min. 1920px breit
- **Hauptbilder:** min. 1200px breit
- **Dateigröße:** max. 500 KB (komprimiert)

## 🔗 Bilder komprimieren

Vor dem Hochladen bitte komprimieren:
- https://tinypng.com
- https://squoosh.app

## ✅ Nach dem Hochladen

Dem Entwickler Bescheid geben, damit die Bilder in die Website eingebunden werden!
